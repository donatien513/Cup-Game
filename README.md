## Installation

Use the package manager [npm](https://www.npmjs.com/) to install this game.

```bash
git clone https://github.com/donatien513/Cup-Game
cd Cup-Game
npm install
```

## Usage

```bash
npm run dev
```

## Test

```bash
npm run test
```

## Build

```bash
npm run build && npm run export
```

Open on your browser http://localhost:3000
